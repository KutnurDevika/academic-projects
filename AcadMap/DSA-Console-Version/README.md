# Academic Roadmap Dashboard – DSA Console Version

## Overview

The DSA Console Version of the Academic Roadmap Dashboard is a simplified implementation of the project using a command-line interface. This version demonstrates how core Data Structures can be used to manage academic tasks without a graphical user interface.

The program allows users to add, view, and manage academic tasks through a menu-driven console application.

## Objective

The goal of this implementation is to demonstrate the logical backend of the Academic Roadmap Planning Dashboard using Data Structures and algorithms.

## Technologies Used

* Java
* Data Structures concepts
* Console-based interaction

## Data Structures Used

* **ArrayList** – Stores academic tasks dynamically
* **Menu-driven logic** – Allows users to perform operations like adding and viewing tasks

## Features

* Add academic tasks
* View planned tasks
* Delete completed tasks
* Simple menu-driven interface
* Demonstrates data structure operations

## Program Workflow

1. The program displays a menu in the console.
2. Users select an option to perform operations.
3. Tasks are stored in a dynamic list.
4. Users can view or remove tasks based on their progress.

## Learning Outcome

This console application demonstrates how Data Structures can be applied to manage and organize academic planning tasks logically. It focuses on backend functionality while the web version focuses on user interface and visualization.

## Relation to Web Version

This console-based implementation represents the logical structure of the Academic Roadmap Dashboard project, while the web version provides an interactive dashboard using HTML, CSS, and JavaScript.
